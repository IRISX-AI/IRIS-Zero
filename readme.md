# IRIS Zero

## About

IRIS Zero is a local AI assistant built with React, TypeScript, and Ollama.

## Features

- Local AI assistant
- Voice assistant
- Can run applications
- Can answer questions
- Can perform various tasks

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Ollama
- Node.js
- Express.js

## Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
